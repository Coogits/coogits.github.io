    Linear Function

    A linear function is a function that represents a straight line[1]Line (Analytic Geometry) in a Cartesian coordinate system.

    Symbols

    • $a\quad$ is the slope (rate of change), determining the steepness of the line.
    • $b\quad$ is the y-intercept, representing the value of the function when $x=0$.

    Formulas

    • Definition:
      Is defined as: $$f(x)=ax+b$$

    Properties

    • The graph is always a straight line.
    • The function has a constant rate of change (slope $a$).
    • The derivative of a linear function is the constant $a$
    • If $a=0$, the function reduces to a constant function[2]Constant Function $f(x)=b$.
    • It is one-to-one if $a\neq0$, meaning it has an inverse.

    Special Cases

    • Relation to Constant Function
      If $a=0$, the function reduces to a constant function
    • Relation to Identity Function if $a=1$ and $b=0$, the function reduces to a identity function

    Calculus

    • Derivative:
      The derivative of a linear function is the constant $a$: $$f'(x)=a$$
    • Antiderivative (Indefinite Integral & Primitive):
      The primitive (antiderivative) of a linear function is: $$F(x)=\frac{1}{2}ax^{2}+bx+C$$ where $C$ is the constant of integration.
