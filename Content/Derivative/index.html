    Derivative (Calculus)

    The derivative of a function represents the rate at which the function's value changes as its input changes. In simpler terms, it measures the instantaneous rate of change of a function with respect to one of its variables. The derivative can be thought of as the slope of the tangent line to the graph of the function at a given point.

    Symbols

    • $f(x_0)$ Function
    • $f'(x_0)\quad$ Is the Derivate of $f(x_0)$
    • $\Delta x\quad$ Is a small change in $x_0$
    • $f(x_0+h)-f(x_0)\quad$ Difference quotient, Is the change in the function value.
    • $x\quad$ is the independent variable.
    • $y\quad$ is the dependent variable (the function being differentiated).
    • $\partial\quad$ replaces $d$ in the Leibniz notation, indicating differentiation with respect to one variable while keeping others constant.

    Definition

    • Formal Definition (using Limits):
      The derivative of a function $f(x)$ at a point $x_0$ is defined as the limit of the average rate of change of the function over an interval as the interval approaches zero. This is written as: $$f'(x)=\displaystyle\lim_{h\to x_0}\cfrac{f(x+h)-f(x)}{h}$$

    Interpretation

    Geometric Interpretation:

    • Slope:
      The first derivative represents the slope[1]Slope (Analytic Geometry) of the tangent line to the curve $y=f(x)$ at a given point.
    • Slope Interpretation : based on the the derivative value, the slope can be:
      • $f'(x)>0\quad$ the function is increasing.
      • $f'(x)>0\quad$ the function is decreasing.
      • $f'(x)=0\quad$ the function may have a critical point (maximum, minimum, or saddle point).

    Properties

    • Constant Function ($f'(x)=0$):
      If $\cfrac{dy}{dx}=0$ for all $x$ in a interval, then $f(x)$ is constant in the interval
    • Non-Differentiable Points:
      A function can be non-differentiable at points where:
      • It has a corner (cusp).
      • It has a discontinuity.
      • It has an inflection point with a vertical tangent.
      • The derivative is not defined at that point.
    • If the limit of the difference quotient exists and is finite, the function $y=f(x)$ is said to be differentiable at the point $x_0$ and the value of the limit is called the derivative of $f(x)$ at $x_0$

    Notations

    • Lagrange notation
      One of the most common modern notations: $$f'(x)$$
    • Leibniz notation
      Expresses derivatives in a way that explicitly emphasizes the variable with respect to which differentiation is performed, represents the rate of change of y with respect to x: $$\cfrac{dy}{dx}$$
    • Partial Derivative Notation:
      Similar to the Leibniz notation, For functions of several variables, where $\partial$ replace $d$, the derivative is often written as: $$\cfrac{\partial}{\partial x}$$

    Higher-Order Derivatives

    The first derivative $f'(x)$ gives the rate of change. We can take derivatives multiple times:

    • Second derivative $f''(x)$ describes acceleration or concavity.
    • Third derivative $f'''(x)$ can be used in physics for jerk (rate of change of acceleration).
    • Higher derivatives are written as$f^{(n)}(x)$.

    Partial Derivative

    A partial derivative measures how a function changes as one of its variables changes while all other variables are held constant. It is used for functions of several variables. is often written as: $$\cfrac{\partial}{\partial x}$$
