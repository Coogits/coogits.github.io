    Mass Number

    The Mass Number, symbolized as A, is the total number of protons and neutrons in the nucleus of an atom.
    It represents the mass number and approximates the atom’s mass in atomic mass units (u).

    Symbols

    • $A$ — Mass Number
    • $Z$ — Atomic Number
    • $N$ — Number of Neutrons

    Formulas and Notations

    • Notation The atomic mass is written as a superscript, with the atomic number as a subscript, before the element symbol:: $$_Z^A\text{ Element Symbol}$$ Example (Carbon): $$_{6}^{12}C$$
    • Mass Number:
      Mass number is the sum of Protons (Z) and Neutrons (N): $$Z=A+N$$

    Properties

    • Isotopes:
      The mass number reflects the isotope of an element.
      • Atoms of the same element can have different atomic masses — these are called isotopes.
    • NOT average atomic weight:
      It is not the same as the average atomic weight shown on the periodic table (which is a weighted average of isotopes).
