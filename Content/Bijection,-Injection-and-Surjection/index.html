    Injective, Surjective, and Bijective Functions

    A function $f: X \to Y$ describes a mapping from a set $X$ (domain) to a set $Y$ (codomain). Depending on how elements in $X$ are assigned to elements in $Y$, a function can be classified as injective, surjective, or bijective.

    Injective Function (One-to-One)

    • Definition
      A function $f: X \to Y$ is injective (or one-to-one) if different inputs map to different outputs. That is: $$f(x_1)=f(x2)\implies x_1=x_2$$

    • Graphically:
      The function passes the horizontal line test, no horizontal line intersects the function more than once.

    • Image Description:
      A Injective (non-Surjective) Function

    Surjective Function (Onto)

    • Definition:
      A function $f: X \to Y$ is surjective if its image (range) is equal to its codomain: $$Im(f)=Y$$

    • Graphically:
      The function's range must be equal to its codomain.

    • Image Description:
      A Surjective (non-Injective) Function

    Bijective Function (One-to-One & Onto)

    • Definition:
      A function $f: X \to Y$ is bijective if it is both injective and surjective.
      This means:
      1. Injective: Different inputs map to different outputs.
      2. Surjective: Every element in the codomain has at least one preimage.
    • Graphically:
      The function passes both the horizontal and vertical line tests.
    • Properties:
      A bijective function has a unique inverse: $$f^{-1}(x)$$
    • Image Description:
      A Bijective (Injective and Surjective) Function

    Non-Injective & Non-Surjective Function

    • Definition
      A function $f: X \to Y$ is neither injective nor surjective if:
      1. It is not injective: Some distinct elements in the domain map to the same element in the codomain.
      2. It is not surjective: Some elements in the codomain are never mapped by any element in the domain.
    • Graphically:
      • The function fails the horizontal line test: Not injective.
      • The function does not cover the entire codomain: Not surjective.
    • Image Description:
      A function that is neither injective nor surjective.
