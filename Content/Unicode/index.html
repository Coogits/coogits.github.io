    Unicode

    Unicode is a universal character encoding standard designed to represent text from all writing systems used worldwide. It assigns a unique numerical code, known as a code point, to each character, ensuring consistent representation across different platforms, languages, and devices.

    • Universal Character Set:
      Covers characters from all languages, including Latin, Cyrillic, Chinese, Arabic, and emoji.
    • Code Points:
      Each character is assigned a unique Unicode code point, written as U+XXXX (e.g., 'A' is U+0041).
    • Backward Compatibility:
      Includes ASCII (U+0000 to U+007F) as a subset.
    • Standardized Across Systems:
      Ensures text is displayed correctly across different computers and applications.

    Unicode Encoding Formats

    Since Unicode code points are abstract numbers, they need to be stored in memory using different encoding formats:

    • UTF-8 (8-bit Unicode Transformation Format)

      • Variable-length encoding (1-4 bytes per character).
      • Backward-compatible with ASCII.
      • Efficient for English and other Latin-based texts.
      • Commonly used in web pages, databases, and modern applications.

    • UTF-16 (16-bit Unicode Transformation Format)

      • Variable-length encoding (2 or 4 bytes per character).
      • Used in Windows, Java, and older Unicode implementations.
      • Supports surrogate pairs for characters outside the Basic Multilingual Plane (BMP).

    • UTF-32 (32-bit Unicode Transformation Format)

      • Fixed-length encoding (always 4 bytes per character).
      • Simple but memory-intensive.
      • Rarely used except in specialized applications.

    Unicode Planes and Ranges

    Unicode organizes characters into planes, each containing $65,536 (2^{16})$ code points:

    • Basic Multilingual Plane (BMP) (U+0000 to U+FFFF)

      • Includes most commonly used characters.
      • Covers Latin, Greek, Cyrillic, Arabic, and CJK (Chinese, Japanese, Korean).

    • Supplementary Planes (U+010000 to U+10FFFF)

      • Includes less common characters, mathematical symbols, historic scripts, and emoji.
