    Programming Language

    A programming language is a formal system of syntax and semantics used to write instructions that a computer can execute. It provides a structured way to define computations, manipulate data, and control hardware or software behavior.

    Concepts

    • Syntax:
      Defines the structure and rules for writing code.

    • Semantics:
      Specifies the meaning of correctly written programs.

    • Execution Model:
      Determines how the written code runs (compiled, interpreted, or hybrid).

    • Paradigm:
      Supports different programming styles like procedural, object-oriented, or functional.

    Language Types

    Compiled Language

    Translated into machine code before execution.

    • C
    • C++
    • C#
    • Java

    Interpreted Language

    Executed line by line at runtime

    • Python
    • JavaScript

    Scripting Language

    Used for automating tasks, often interpreted

    • Bash
    • PHP
    • Lua

    Markup Language

    Define structure or appearance rather than logic

    • Html
    • Xml
    • Markdown
    • Latex
    • CSS
