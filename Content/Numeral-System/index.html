    Numeral System

    A numeral system is a writing system for expressing numbers. It's a set of symbols and rules for using those symbols to represent numerical values. Any system can be converted in any other.

    Symbols

    • $N\quad$ Represent the number.
    • $a_i\quad$ Represent the digits of the number at position $i$.
    • $b^i\quad$ Represent the digit's positional weight.
    • $i\quad$ Represent the position from right to left with $0$ as the first position before the comma.
    • $b\quad$ Represent the base of the Numeral System.

    Notations

    • Positional Notation:
      Where $a$ represent the digit based on it's position. $$\ldots a_1a_0,a_{-1}a_{-2}\ldots \qquad 0\leq a<b$$
    • Base Notation:
      The base is specified as pedice of the number (i.e. one hundred $100_{10}$, one zero zero $100_2$) $$N_b$$
    • Expanded Form (Numeral System):
      is a method of representing numbers in which the position of each digit within the number determines its value, based on powers of a fixed base. $$N=\displaystyle\sum_{i=-m}^na_i\cdot b^i$$

    Common Numeral Systems

    Binary System

    It uses only two digits.

    Octal System

    The octal system (also called base 8 numeral system) is a positional numeral system that uses eight digits to represent numbers.

    Decimal System

    The most common system

    Hexadecimal System

    The hexadecimal system (also known as base-16) is a numeral system, that uses 16 symbols to represent values.
