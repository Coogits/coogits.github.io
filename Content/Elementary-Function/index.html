    Elementary Function

    An elementary function is a Function composed of basic functions that can be expressed using a finite number of standard operations, such as addition, subtraction, multiplication, division, and composition. These functions include polynomial, rational, exponential, logarithmic, trigonometric, and their inverses. They are fundamental to calculus and other branches of mathematics because they can describe a wide range of phenomena and are well-understood in terms of limits, derivatives, and integrals.

    Most Common Functions

    Constant Function

    A function that always returns the same value, regardless of x: $$f(x)=k$$

    Linear Function

    Represents a straight-line relationship with a constant rate of change (slope). $$f(x)=ax+b$$

    Identity Function

    A special case of the linear function where the output equals the input. $$f(x)=x$$

    Absolute Value Function

    Measures the distance of xxx from zero, creating a V-shaped graph. $$f(x)=\lvert x \rvert$$

    Polynomial Function

    A function composed of terms with non-negative integer exponents, forming curves of varying degrees. $$f(x)=a_nx^{n}+a_{n-1}x^{n-1}+\ldots+a_{2}x^{2}+a_{1}x^{1}+a_{0}$$

    Reciprocal Function

    A rational function where the numerator is 1 and the denominator is a linear monomial: $$f(x)=\cfrac{1}{x}$$

    Rational Function

    A ratio of two polynomials, potentially introducing asymptotes. $$f(x)=\dfrac{P(x)}{Q(x)}$$

    Power Function

    A function where the variable xxx is raised to a fixed real exponent. $$f(x)=x^n \quad n\in\mathbb{R}\text{ constant}$$

    Exponential Function

    Models rapid growth or decay, where the variable appears as an exponent. $$f(x)=a^x\qquad a>0\land a\neq0$$

    Logarithmic Function

    The inverse of the exponential function, used for solving exponential equations. $$f(x)=\log_a x\qquad a,x>0;a\neq1$$

    Root Function

    Represents the inverse of a power function, defining roots of a number. $$f(x)=\sqrt[n]{a} \qquad n\in\mathbb{N}\backslash\{0\}, a\in\mathbb{R}$$

    Trigonometric Functions

    Sine Function

    Describes periodic wave behavior. $$f(x)=\sin⁡ x$$

    Cosine Function

    Similar to sine but phase-shifted by π2\frac{\pi}{2}2π​. $$f(x)=\cos⁡x$$

    Tangent Function

    Represents the ratio of sine to cosine. $$f(x)=\tan ⁡x=\cfrac{\sin x}{\cos x}$$

    Cotangent Function

    The reciprocal of the tangent function. $$f(x)=\cot ⁡x=\cfrac{\cos x}{\sin x}$$

    Secant Function

    The reciprocal of the cosine function. $$f(x)=\sec ⁡x=\cfrac{1 x}{\cos x}$$

    Cosecant Function

    The reciprocal of the sine function. $$f(x)=\csc ⁡x=\cfrac{1 x}{\sin x}$$
