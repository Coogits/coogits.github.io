    Stack

    A stack is a linear data structure that follows the Last In, First Out (LIFO) principle, meaning the last element added is the first one to be removed. It is analogous to a stack of plates where the topmost plate is removed first.

    Key Points

    • LIFO:
      Last In, First Out, meaning the last element added is the first one to be removed
    • Time Complexity:
      Stack operations (push, pop, peek, isEmpty) run in O(1) time since they only modify or access the top element.

    Implementation

    Stacks can be implemented using:

    Arrays

    Fixed-size stack with a predefined limit.
    ✅ Pros: Fast, simple, uses contiguous memory.
    ❌ Cons: Fixed size, cannot grow dynamically.
    Example:

    #include <iostream>
using namespace std;

class MyStack {
private:
    int* stack;
    int top;
    int maxSize;
public:
    MyStack(int size) {
        maxSize = size;
        stack = new int[maxSize];
        top = -1;
    }
    void push(int value) {
        if (top == maxSize - 1) {
            cout << "Stack Overflow\n";
            return;
        }
        stack[++top] = value;
    }
    int pop() {
        if (top == -1) {
            cout << "Stack Underflow\n";
            return -1;
        }
        return stack[top--];
    }
    int peek() {
        if (top == -1) {
            cout << "Stack is empty\n";
            return -1;
        }
        return stack[top];
    }
    bool isEmpty() {
        return top == -1;
    }
    ~MyStack() {
        delete[] stack;
    }
};
// Example Usage
int main() {
    MyStack s(5);
    s.push(10);
    s.push(20);
    cout << "Top element: " << s.peek() << endl; // Output: 20
    cout << "Popped: " << s.pop() << endl;       // Output: 20
    return 0;
}

    Linked Lists

    Dynamic stack without a size constraint.
    ✅ Pros: No fixed size, flexible.
    ❌ Cons: Extra memory for pointers, slower due to dynamic allocation.
    Example:

    #include <iostream>
using namespace std;

class MyStack {
private:
    struct Node {
        int data;
        Node* next;
    };
    Node* top;
public:
    MyStack() {
        top = nullptr;
    }
    void push(int value) {
        Node* newNode = new Node{value, top};
        top = newNode;
    }
    int pop() {
        if (!top) {
            cout << "Stack Underflow\n";
            return -1;
        }
        Node* temp = top;
        int value = temp->data;
        top = top->next;
        delete temp;
        return value;
    }
    int peek() {
        return top ? top->data : -1;
    }
    bool isEmpty() {
        return top == nullptr;
    }
    ~MyStack() {
        while (top) pop();
    }
};
// Example Usage
int main() {
    MyStack s;
    s.push(30);
    s.push(40);
    cout << "Top element: " << s.peek() << endl; // Output: 40
    cout << "Popped: " << s.pop() << endl;       // Output: 40
    return 0;
}

    Comparison

    Feature Array Stack Linked List Stack
    Memory Usage Fixed, no extra pointers Extra memory for pointers
    Performance Faster (O(1) push/pop) Slightly slower due to allocation
    Size Limit Fixed (predefined) Dynamic (grows as needed)
    Implementation Simple, easy to use Requires additional memory management

    Operations

    A stack supports the following primary operations:

    Push(X)

    Adds element X to the top of the stack.

    • If the stack is full (in array implementation), an overflow occurs.
    • If using a linked list, no fixed limit exists. Example with Array Implementation
    void push(int stack[], int &top, int X, int maxSize) {
    if (top == maxSize - 1) {
        cout << "Stack Overflow\n";
        return;
    }
    stack[++top] = X;
}

    Example with Linked List Implementation

    struct Node {
    int data;
    Node* next;
};
void push(Node*& top, int X) {
    Node* newNode = new Node{X, top};
    top = newNode;
}

    Pop()

    Removes and returns the top element of the stack.

    • If the stack is empty, an underflow occurs. Example with Array Implementation
    int pop(int stack[], int &top) {
    if (top == -1) {
        cout << "Stack Underflow\n";
        return -1;
    }
    return stack[top--];
}

    Example with Linked List Implementation

    int pop(Node*& top) {
    if (!top) {
        cout << "Stack Underflow\n";
        return -1;
    }
    Node* temp = top;
    int value = temp->data;
    top = top->next;
    delete temp;
    return value;
}

    Peek()

    Peek() or Top() returns the top element without removing it.
    Example with Array Implementation

    int peek(int stack[], int top) {
    if (top == -1) {
        cout << "Stack is empty\n";
        return -1;
    }
    return stack[top];
}

    Example with Linked List Implementation

    int peek(Node* top) {
    if (!top) {
        cout << "Stack is empty\n";
        return -1;
    }
    return top->data;
}

    IsEmpty()

    Checks if the stack is empty.
    Example with Array Implementation

    bool isEmpty(int top) {
    return top == -1;
}

    Example with Linked List Implementation

    bool isEmpty(Node* top) {
    return top == nullptr;
}

    Applications

    • Function call management (recursion, backtracking).
    • Undo/Redo functionality in text editors.
    • Expression evaluation (infix to postfix conversion, balancing parentheses).
    • Depth-First Search (DFS) in graph algorithms.
