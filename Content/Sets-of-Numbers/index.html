    Sets of Numbers

    A set of numbers is a well-defined collection of numeric elements, grouped based on specific properties. Each number belongs to one or more sets depending on its characteristics.

    Main Sets of Numbers

    • Natural Number Set ($\mathbb{N}$):
      The set of positive integers (whole numbers greater than zero): $$\mathbb{N}=\{1,2,\ldots,+\infty\}$$
    • Whole Number Set ($\mathbb{N}_0$):
      The set of natural numbers including zero: $$\mathbb{N}_0=\mathbb{N}\cup\{0\}$$
    • Integer Number Set ($\mathbb{Z}$):
      The set of whole numbers and their negatives: $$\mathbb{Z}=\mathbb{N}^-\cup\mathbb{N_0}$$
    • Rational Number ($\mathbb{Q}$):
      The set of all numbers that can be expressed as a fraction $\cfrac{a}{b}$ where 'a' and 'b' are integers and 'b' is not zero. $$\begin{align}\mathbb{Q}=\cfrac{a}{b} \qquad\qquad \\ \text{where }a,b\in\mathbb{Z};b\neq0\end{align}$$
    • Real Number ($\mathbb{R}$):
      The set of all rational and irrational numbers (numbers that cannot be expressed as a fraction, such as $\pi$ and $\sqrt2$). $$\mathbb{R}=\{x:-\infty<x<\infty\}$$
    • Complex Numbers ($\mathbb{C}$):
      The set of numbers that can be expressed in the form $a+ib$, where 'a' and 'b' are real numbers, and 'i' is the imaginary unit ($i=\sqrt{-1}$).
