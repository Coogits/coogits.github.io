    Operator (Programming)

    A programming operator is a symbol or function that performs a specific operation on one or more operands (values or variables) in a program. Operators are fundamental components of programming languages and are used to manipulate data, perform calculations, and control program flow.

    Operator Types

    Programming operator types:

    Unary Operator

    A unary operator is an operator that operates on a single operand to perform an operation.
    Unary operators are useful for manipulating variables efficiently and are commonly used in loops, conditions, and memory operations.

    • Increment/Decrement Operators:
      Modify a value by 1:
      • ++ (Increment)
      • -- (Decrement)
    • Negation Operators
      • Arithmetic negation: -x (Changes the sign of a number)
      • Logical negation: !x (Flips Boolean values)
    • Type Operators (Language-Specific) Used for type checking and conversion:
      • typeof (JavaScript)
      • instanceof (JavaScript, Java)
      • as (TypeScript, C#)
    • Pointer Operators (C, C++) Work with memory addresses:
      • * (Dereference)
      • & (Address-of)

    Binary Operator

    A binary operator is an operator that requires two operands to perform an operation.
    It takes one operand on the left and another on the right.
    Binary operators are fundamental in programming and are used for computations, comparisons, and logical operations.

    • Arithmetic Operators
      Perform mathematical calculations:
      • + (Addition)
      • - (Subtraction)
      • * (Multiplication)
      • / (Division)
      • % (Modulus - Remainder of division)
    • Relational (Comparison) Operators
      Compare values and return a Boolean result:
      • == (Equal to)
      • != (Not equal to)
      • > (Greater than)
      • < (Less than)
      • >= (Greater than or equal to)
      • <= (Less than or equal to)
    • Logical Operators
      Perform Boolean logic:
      • && (Logical AND)
      • || (Logical OR)
      • ! (Logical NOT, technically unary)
    • Bitwise Operator
      Operate at the binary level:
      • & (Bitwise AND)
      • | (Bitwise OR)
      • ^ (Bitwise XOR)
      • ~ (Bitwise NOT, technically unary)
      • << (Left shift)
      • >> (Right shift)
    • Assignment Operators
      Assign or update values efficiently:
      • = (Assignment)
      • += (Addition assignment)
      • -= (Subtraction assignment)
      • *= (Multiplication assignment)
      • /= (Division assignment)

    Ternary Operator

    Shorthand for conditional expressions

    • condition ? true_value : false_value (C/C++)

    Operator Overloading

    Some programming languages, like C++ and Python, allow operator overloading, where operators can be redefined to perform different operations based on the types of operands involved.
    This allows operators to be used with user-defined types, such as classes or structs, in a way that mimics built-in types.

    Assignment Operator
    An operator used to assign a value to a variable, such as the assignment (=) operator. It can also include compound assignment operators like +=, -=, *=, etc.

    Logical and Relational Operators
    These include operators for comparison and logical operations, such as && (AND), || (OR), > (greater than), <= (less than or equal to), etc.

    Bitwise Operators
    Operators that perform bit-level operations on integer values, such as AND (&), OR (|), XOR (^), and shift operators (<<, >>).

    Precedence and Associativity
    Operators also have precedence (the order in which operations are performed) and associativity (whether operations are performed left-to-right or right-to-left), which dictate how expressions are evaluated.
