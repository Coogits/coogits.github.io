    Combinatorics

    Combinatorics is the branch of mathematics that studies the counting, arrangement, and combination of objects. It provides the foundational tools and techniques for analyzing the possible outcomes in probabilistic scenarios. Combinatorics helps determine the total number of possible outcomes (sample spaces) and the number of favorable outcomes (events) in various probability problems.

    Key concepts

    • Binomial Coefficient:
      Often used in probability theory, especially in the study of binomial distributions, the binomial coefficient represents the number of ways to choose $k$ successes in $n$ independent Bernoulli trials: $$\displaystyle\binom{n}{k}=\cfrac{n!}{k!(n-k)!}\qquad n,k\in\mathbb{N}\quad n\geq k$$

    • Permutation:
      The number of ways to arrange $n$ distinct objects in a specific order: $$P_n=n!$$

    • Dispositions:
      The number of ways to choose a subset of $k$ objects from a set of $n$ distinct objects, where the order matters: $$D_{n,k}=\cfrac{n!}{(n-k)!}$$

    • Combinations:
      The number of ways to choose a subset of $k$ objects from a set of $n$ distinct objects, without regard to the order of selection: $$C_{n,k}=\displaystyle\binom{n}{k}=\cfrac{n!}{k!(n-k)!}$$

    • Counting Principles:
      Fundamental counting principles like the multiplication rule (if one event can happen in $m$ ways and a second event can happen independently in $n$ ways, the total number of ways both events can happen is $m\times n$).
