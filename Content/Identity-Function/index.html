    Identity Function

    The identity function is a function that returns its input unchanged.

    Symbols

    • $f()\quad$ Function notation
    • $x\quad$ Variable

    Formulas

    • Definition:
      Defined as: $$f(x)=x$$

    Properties

    • The graph is a straight line passing through the origin with a slope of 1 (i.e., the bisector of the first and third quadrants).
    • The function is bijective, meaning it is both one-to-one and onto.
    • The derivative is always 1:
    • It is the neutral element for function composition: for any function $g(x)$, $$g(f(x))=g(x)\text{ and }f(g(x))=g(x)$$

    Calculus

    • Derivative:
      The derivative is always 1: $$f'(x)=1$$
    • Primitive:
      The primitive (antiderivative) is: $$F(x)=\cfrac{x^{2}}{2}$$

    Special Cases

    • Relation to Linear Function:
      A linear function can be viewed as a identity function where the slope $a=1$ and $b=0$: $$f(x)=1x+0$$
