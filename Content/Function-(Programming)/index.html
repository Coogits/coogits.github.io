    Function (Programming)

    A function is a self-contained block of code that performs a specific task and can be reused throughout a program. Functions help organize code, improve readability, and promote reusability by allowing a piece of logic to be executed multiple times without repetition.

    Concepts

    • Identifier:
      A function typically has a name that identifies it.

    • Parameters (Inputs):
      Functions may accept input values (arguments) to modify their behavior.

    • Return Value (Output):
      A function may return a value after execution.

    • Encapsulation:
      Functions encapsulate logic, preventing interference with other parts of the program.

    • Reusability:
      Functions can be called multiple times, reducing code duplication.

    • Modularity:
      Functions help break down complex programs into smaller, manageable pieces.

    Structure

    Most programming languages define functions with:

    1. declaration (optional in some languages).
    2. definition containing the function body.
    3. return type (or void if no return value is needed).
    4. Parameters (optional).
    5. Control flow with subroutine to execute it.

    Examples

    // Declaration ("add" is identifier; "a,b" are parameters; "int" the return type)
int add(int a, int b);
// 
int main() {
    int result = add(5, 3);  // Function call
    return 0;
}
// Definition
int add(int a, int b) { // encapsulation done via brackets {}
    return a + b;
}

    def add(a, b):
    return a + b
result = add(5, 3)  # Calling the function

    function add(a, b) {
    return a + b;
}
let result = add(5, 3);  // Calling the function

    Parameter

    A parameter is a variable listed in a function's declaration or definition that receives a value (known as an argument) when the function is called. Parameters allow functions to accept inputs and operate on them dynamically.

    • Formal Parameters:
      Defined in the function signature and act as placeholders.
    • Actual Arguments
      Values passed to the function during a call.
    • Pass by Value:
      The function gets a copy of the argument.
    • Pass by Reference:
      The function gets a reference to the original variable.
    • Default Parameters:
      Parameters that have a default value if no argument is provided.
    • Variadic Parameters:
      Allow functions to accept a variable number of arguments.
