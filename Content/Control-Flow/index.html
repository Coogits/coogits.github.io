    Control Flow (Programming)

    Control flow refers to the order in which individual statements, instructions, or function calls are executed within a program. It determines how the program progresses from one operation to another based on conditions, loops, and function calls.

    Types of Control Flow Constructs

    Sequential Execution

    • The default execution model where statements are executed one after another in the order they appear.
    # Example (Python)
print("Step 1")
print("Step 2")
print("Step 3")

    Branching

    • It is conditional execution, meaning the control flow can follow different paths based on conditions.
    • Allows a program to make decisions dynamically (e.g., if, else, switch).
    • Unlike jumps, branching evaluates a condition before selecting an execution path.
    // Example (C++)
int x = 10;
if (x > 5) {
    std::cout << "x is greater than 5";
} else {
    std::cout << "x is 5 or less";
}

    Loops

    • Repeats a block of code multiple times based on a condition (for, while, do-while).
    // Example (Java - 'for' loop)
for (let i = 0; i < 5; i++) {
    console.log("Iteration: " + i);
}

    Subroutine

    • A subroutine is a self-contained block of code that performs a specific task.
    • It can be invoked through a function call or another mechanism, depending on the programming language.
    • Control transfers to the function, executes its instructions, and then returns to the caller.
    • May return a value (function) or simply execute instructions (procedure).
    // Example (C)
void greet() {
    printf("Hello, world!");
}
int main() {
    greet(); // Function call
    return 0;
}

    Exception Handling

    • Used to handle errors and unexpected conditions (try-catch, throw).
    # Example (Python)
try:
    result = 10 / 0  # Division by zero error
except ZeroDivisionError:
    print("Cannot divide by zero!")

    Jump

    • Break: Exits a loop or switch statement.
    • Continue: Skips the rest of the loop iteration.
    • Goto: (Discouraged) Jumps to a labeled statement.
    # Example (`break` in C++):
for (int i = 0; i < 10; i++) {
    if (i == 5) break; // Exit loop when i reaches 5
    std::cout << i << " ";
}
