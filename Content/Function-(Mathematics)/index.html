    Function (Mathematics)

    A function is a mathematical relation that associates each element of a set (called the domain) to exactly one element of another set (called the codomain or range). In simple terms, a function defines a rule that assigns a unique output to each input.

    In the Image:
    Graphic representation of a function

    Symbols

    • $f()\quad$ Function notation.
    • $X\quad$ Domain, the set in input.
    • $Y\quad$ Codomain, the set in output.
    • $x\quad$ Element from the Domain $A$.
    • $y\quad$ Element from the Codomain $B$.
    • $Im(f)\quad$ Range of the function

    Definition

    • Definition:
      For each $x$ in a set $X$ (Domain) exist only one $y$ in a set $Y$ (Codomain) such that $y$ is equal to $f(x)$:$$\forall x \in X\;\exists!\;y\in Y :\quad y=f(x)$$
    • Formal Definition:
      A function $f$ from a set $X$ (the domain) to a set $Y$ (the codomain) is denoted as: $$f:X\to Y$$

    Key Concepts

    • Domain $(X)$:
      The set of all possible inputs (values of $x$) that the function can accept.
    • Codomain $(Y)$:
      The set of all possible outputs that the function can produce (not necessarily the actual outputs).
    • Range $(Im(f))$:
      The set of all actual outputs that the function produces from the domain.

    Types of Functions

    See Bijection, Injection and Surjection for formal definitions.

    • Injective (One-to-One):
      A function where each element in the domain maps to a unique element in the codomain, and no two different elements in the domain map to the same element in the codomain.
    • Surjective (Onto):
      A function where the image set (range) is equal to the codomain, meaning every element in the codomain has at least one corresponding element in the domain.
    • Bijective:
      A function that is both injective and surjective. Every element in the domain corresponds to exactly one unique element in the codomain, and every element in the codomain has exactly one element in the domain.

    Elementary Function

    An elementary function is a Function composed of basic functions that can be expressed using a finite number of standard operations, such as addition, subtraction, multiplication, division, and composition.

    • Constant Function

    Calculus

    In Calculus, a function describes how quantities change and interact. Two fundamental concepts in calculus are limits and continuity, which establish the basis for differentiation and integration.

    Limit (Mathematics)

    Limits form the foundation of calculus, defining how functions behave as inputs approach specific values. They help define derivatives and integrals rigorously.

    Continuity

    Continuity ensures that functions behave smoothly without jumps, breaks, or holes. A function is continuous if its limit at a given point equals its function value, meaning it can be drawn without lifting a pencil.
