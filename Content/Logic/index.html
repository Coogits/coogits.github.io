    Logic

    Logic is the formal study of principles governing valid reasoning, inference, and truth-preserving transformations.
    It provides a structured framework for evaluating the correctness of arguments and is foundational in mathematics, computer science, philosophy, and artificial intelligence.
    Logic deals with the principles of correct reasoning and the structure of statements and arguments.

    Core

    Propositions

    A proposition is a declarative statement that is either true (1) or false (0)—but not both. This binary nature aligns directly with bits, which also have two possible states.

    • Example: "The sky is blue."
      • This is a proposition because it can be evaluated as either true (1) or false (0).
      • Similarly, a bit stores a value of 1 (true) or 0 (false) in digital systems. Thus, propositions in logic and bits in computing share the same fundamental concept of binary states, forming the basis for Boolean algebra, logical reasoning, and digital computation.

    Boolean Algebra

    Boolean algebra is the foundation of logic circuits and binary computation, using a set of logical operations on binary values (0 and 1).

    • Operators: AND, OR, XOR, XAND, NAND, NOR, NOT
    • Fundamental Laws: Identity, Domination, Idempotent, Complement, and De Morgan’s Laws

    Connectives

    Logical operators define how propositions combine and interact. In Boolean Algebra, these operators are used to manipulate binary values (0 and 1), forming the foundation of logical reasoning and digital logic design.

    • IMPLIES (→): True unless the first proposition is true and the second is false.
    • IF AND ONLY IF (↔): True if both propositions have the same truth value.

    Truth Tables

    A table that shows all possible truth values of a compound proposition based on the truth values of its components.

    • Example: Truth table for $A\land B$: $$\begin{array}{|c|c|c|}\hline A&B&A\land B\\\hline 0&0&0 \\\hline 0&1&0 \\\hline 1&0&0 \\\hline 1&1&1\\\hline \end{array}$$

    Logical Equivalence

    Two propositions are logically equivalent if they have the same truth value in all possible scenarios.

    • Example: $P\to Q$ is logically equivalent to $\neg (P\lor Q)$.

    Tautology, Contradiction

    • A tautology is a proposition that is always true (e.g., $P\lor\neg P$).
    • A contradiction is a proposition that is always false (e.g., $P\land\neg P$).

    Quantifiers

    Used in predicate logic to express statements about "all" or "some" elements in a domain.

    • Universal Quantifier (∀): "For all" or "For every."
    • Existential Quantifier (∃): "There exists" or "For some."

    Arguments and Validity

    • An argument consists of premises and a conclusion.
    • An argument is valid if the conclusion logically follows from the premises.

    Types of Logic

    • Propositional Logic:
      • Deals with simple propositions and their combinations using logical connectives.
    • Predicate Logic (First-Order Logic):
      Extends propositional logic by introducing quantifiers, predicates, and variables.
    • Modal Logic:
      Extends classical logic to include modalities like "necessarily" and "possibly."
    • Fuzzy Logic:
      Allows for truth values between 0 and 1 (useful for dealing with uncertainty).
    • Temporal Logic:
      Deals with propositions qualified in terms of time (e.g., "always," "eventually").
