    Database Management System (DBMS)

    A Database Management System (DBMS) is software that enables users to create, manage, and manipulate databases efficiently.
    It provides tools to store, retrieve, modify, and secure data while ensuring integrity and consistency.

    Key Functions of a DBMS

    1. Data Storage & Retrieval – Organizes data efficiently for quick access.
    2. Data Security – Controls user access and protects data from unauthorized use.
    3. Data Integrity – Maintains accuracy, consistency, and reliability of data.
    4. Concurrency Control – Allows multiple users to access data simultaneously without conflicts.
    5. Backup & Recovery – Prevents data loss by providing recovery mechanisms.

    Relational Database

    Relational DBMS (RDBMS), Uses tables and structured query language (SQL):

    • MySQL – Open-source RDBMS used in web applications.
    • PostgreSQL – Advanced RDBMS known for extensibility and compliance.
    • SQLite – Lightweight, file-based database for local applications.

    NoSQL Database

    NoSQL DBMS – Designed for unstructured or semi-structured data.

    • MongoDB
    • Cassandra

    Hierarchical DBMS

    Organizes data in a tree-like structure. Example:

    • IBM
    • IMS

    Network DBMS

    Uses a graph structure for relationships. Example:

    • IDMS

    Object-Oriented DBMS (OODBMS)

    Stores data as objects. Example:

    • db4o.
