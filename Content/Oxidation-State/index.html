    Oxidation State

    The oxidation state of an atom in a compound is a number that represents the hypothetical charge the atom would have if all bonds were purely ionic (i.e., if electrons were fully transferred, not shared).

    Process

    Example: H₂O

    1. Start with the molecule:
      Draw the molecular structure using the Lewis structure (total charge is 0): $$H_2O\quad\text{ drawn as }\quad H-\overline{\underline O}-H$$
    2. Break the bonds hypothetically and assign electrons based on electronegativity:
      Oxygen is more electronegative than Hydrogen, so it takes both bonding electrons from each H–O bond. $$H\cdot\;\cdot \overline{\underline O}\cdot\;\cdot H\quad\to\quad H\;\rvert\overline{\underline O}\lvert\;H$$
    3. Calculate resulting charges:
      • Hydrogen normally has 1 electron. Now it has 0 electrons, so it loses 1: $$1-0=1$$
      • Oxygen normally has 6 valence electrons. Now it has 8 electrons, so it gains 2: $$6-8=-2$$
    4. Final oxidation states:
      The result of each difference $N_i-N_f$ (initial minus final electrons) gives the oxidation state: $$H_2^{+1}O^{-2}$$
    5. Verify:
      the sum of all oxidation states equals the total charge of the molecule: $$2\cdot(+1)+(-2)=0$$

    Common Oxidation States

    • Oxygen (O)
      • for oxides: -2
        • e.g. water (H₂O), sodium oxide (Na₂O), perchloric anhydride (Cl₂O₇)
      • for dioxides: -1
        • e.g. hydrogen peroxide (H₂O₂), sodium peroxide (Na₂O)
    • Hydrogen (H)
      • Almost always: +1
        • e.g. H₂OH
      • In metal hydrides (hydride ion, $H^{-}$): -1
        • e.g. sodium hydride (NaH), potassium hydride (KH)
    • Alkali metals (Group 1)
      • Always: +1
        • Examples: $Li^+,Na^+,K^+$
    • Alkaline earth metals (Group 2)
      • Always: +2
        • Examples: $Mg^{2+}, Ca^{2+}, Ba^{2+}$
    • Carbon (C)
      • carbon monoxide (CO): +2
      • Carbon dioxide (CO₂): +4
      • Organic Compound:
        • Depends on the specific bonds — use the electron-counting process above to calculate.
    • Nitrogen (N)
      • multiple values
    • Fluorine (F)
      • Always: -1
    • Halogens
      • Halides: -1
        • e.g. chlorides ($Cl^-$), bromides($Bl^-$), iodides($I^-$)

    Element's Oxidation States

    The oxidation state on elements or elemental molecules is always zero
