    Boolean Algebra

    Boolean Algebra is a branch of algebra that deals with binary variables and logical operations. It is the foundation of digital logic design and is widely used in computer science, electronics, and mathematical logic.

    Binary Digit (Bit)

    Boolean Algebra operates exclusively on binary values, which can take only two possible states:

    • 1 (True)
    • 0 (False)

    These values correspond to high and low voltage levels in digital electronics and true/false logic in computation.

    Logical Operator

    A logical operator is a symbol or function that combines one or more Boolean values (true or false) to produce another Boolean value. These operators are fundamental in logic, mathematics, computer science, and digital circuits, where they are used to construct and evaluate logical expressions.

    Common logical operators include:

    AND

    • Symbols: ( $\cdot$ or $\land$ )
    • Condition: $A\land B$ is True if both A and B are 1.

    OR

    • Symbols: ( $+$ or $\lor$ )
    • Condition: $A\lor B$ is True if at least one of A or B is 1.

    XOR

    • Symbols: (⊕, ^)
    • Condition: $A ⊕ B$ is True if A and B are different (1 ⊕ 0 = 1, 0 ⊕ 1 = 1), otherwise False.

    XAND

    • Symbols: (⊙, ≡, ↔ )
    • Condition: $A\odot B$ is True if A and B are the same (1 ⊙ 1 = 1, 0 ⊙ 0 = 1), otherwise False.
    • XNOR: Equivalent to NOT (A XOR B).

    NAND

    • Symbols: (↑)
    • Condition: $A \uparrow B$ is True unless both A and B are 1.
    • Not AND: Equivalent to NOT (A AND B).

    NOR

    • Symbols: (↓)
    • Condition: $A \downarrow B$ is True only if both A and B are 0.
    • Not OR: Equivalent to NOT (A OR B).

    NOT

    • Symbols: ( $\neg$ or $\sim$ )
    • Condition: $\neg A$ inverts the value (1 becomes 0, and vice versa).

    Properties

    • Binary Operators
      Most Logical operators(AND, OR, XOR, XNOR, NAND, NOR) are binary operators, meaning they require two operands, The only exception is NOT, which is unary.
    • Commutative
      Most Logical operators (AND, OR, XOR, XNOR, NAND, NOR) are commutative, meaning the order of operands does not change the result: $$A*B=B*A$$
    • NOT is Unary and Non-Commutative
      NOT (¬A) is a unary operator, meaning it only takes one operand, so commutativity does not apply.

    Boolean Theorems

    Fundamental Laws

    Boolean algebra follows a set of fundamental laws:

    • Identity Laws:
      • $A \lor 0 = A$
      • $A \land 1 = A$
    • Domination Laws:
      • $A \lor 1 = 1$
      • $A \land 0 = 0$
    • Idempotent Laws:
      • $A \lor A = A$
      • $A \land A = A$
    • Complement Laws:
      • $A \lor \neg A = 1$
      • $A \land \neg A = 0$

    De Morgan's Theorems

    De Morgan’s Laws define how NOT distributes over AND and OR operations:

    • First Law:
      $\neg(A\land B)=\neg A\lor\neg B$

    • Second Law:
      $\neg(A\lor B)=\neg A\land\neg B$

    These theorems are essential for simplifying logic circuits and Boolean expressions.
