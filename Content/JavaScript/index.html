    JavaScript

    JavaScript (JS) is a high-level, interpreted programming language primarily used to create interactive and dynamic web pages. It is one of the core technologies of the web, alongside HTML and CSS. JavaScript enables developers to manipulate the Document Object Model (DOM), handle events, validate forms, create animations, and communicate with servers asynchronously (AJAX).

    Key Features

    • Lightweight & Interpreted: Runs in the browser without requiring compilation.
    • Dynamically Typed: Variable types are determined at runtime.
    • Prototype-Based Object-Oriented: Uses prototypes instead of traditional class-based inheritance.
    • Event-Driven & Asynchronous: Supports event handling and non-blocking execution (e.g., setTimeout, fetch).
    • Cross-Platform: Runs on all modern web browsers without additional plugins.
    • Supports Functional & Object-Oriented Programming: Allows both paradigms.

    Common Uses

    • Web development (frontend frameworks like React, Vue, Angular)
    • Backend development (Node.js)
    • Game development
    • Mobile app development (React Native)
    • Server-side scripting
    • API handling (RESTful and GraphQL)

    Function

    A function in JavaScript is a block of reusable code designed to perform a specific task. Functions allow code modularity, reusability, and better organization.

    function functionName(parameters) {
    // Code to be executed
    return value; // (optional) return statement
}

    Type of JavaScript Functions:

    Named Function

    A function with a specified name.

    function add(a, b) {
    return a + b;
}
console.log(add(5, 3)); // Output: 8

    Anonymous Function

    A function without a name, often assigned to a variable.

    const multiply = function(a, b) {
    return a * b;
};
console.log(multiply(4, 2)); // Output: 8

    Arrow Function (ES6)

    A shorter syntax for writing functions.

    const subtract = (a, b) => a - b;
console.log(subtract(10, 4)); // Output: 6

    Immediately Invoked Function Expression (IIFE)

    A function that executes immediately after its definition.

    (function() {
    console.log("This runs immediately!");
})();

    Higher-Order Functions

    Functions that accept other functions as arguments or return functions.

    function applyFunction(func, value) {
    return func(value);
}
const double = (num) => num * 2;
console.log(applyFunction(double, 5)); // Output: 10
