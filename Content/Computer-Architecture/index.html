    Computer Architecture

    Computer Architecture is the discipline concerned with the design, structure, and behavior of computer systems. It involves both the theoretical and practical aspects of organizing the hardware to meet specific performance, energy, and cost requirements.

    key Concepts

    • Instruction Set Architecture (ISA):
      The abstract model that defines the supported instructions, data types, registers, addressing modes, and memory models.

    • Microarchitecture:
      The detailed organization of the processor's internal components (such as the ALU, control unit, registers, and cache) that implements the ISA.

    • Architectural Models

      • Von Neumann Architecture
      • Harvard Architecture
      • RISC vs. CISC Architecture

    • Central Processing Unit CPU

      • Arithmetic Logic Unit ALU
      • Control Unit CU

    • Memory Systems

      • Random Access Memory RAM
      • Read Only Memory ROM
      • Memory Hierarchy (including caches, virtual memory, etc.)
