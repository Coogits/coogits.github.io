    From Binary to Decimal Conversion

    Converting a Binary number to a decimal number .

    Symbols

    • $N_2\quad$ Represent a binary number $b=2$.
    • $N_{10}\quad$ Represent the equivalent decimal number.
    • $n\quad$ Represent the most significative digit position ($n\geq0$).
    • $-m\quad$ Represent the least significative digit position ($m\leq0$).
      • $m=0\quad$ No fractional digits.
      • $m>0\quad$ Fractional digits
    • $i\quad$ Represent the position of the digit.
    • $a_i\quad$ Represent the digit of the number at position $i$ ($0\leq a_i< 2$).
    • $b^i\quad$ Represent the positional weight associated with digit $a_i$.

    Formulas

    • From Binary to Decimal:
      Calculate the binary Expanded Form (Numeral System) by multiplying the terms and then summing the resulting terms to obtain the decimal value.:
      • $N_2=\displaystyle\sum_{i=-m}^na_i\cdot 2^i\quad\to\quad N_{10}$

    Programming

    In most programming languages, binary to decimal conversion can be done using simple loops or built-in functions:

    def binary_to_decimal(binary_str):
    decimal = 0
    for i, bit in enumerate(reversed(binary_str)):
        decimal += int(bit) * (2 ** i)
    return decimal

binary = "1101"
print(f"Decimal: {binary_to_decimal(binary)}")  # Output: Decimal: 13
