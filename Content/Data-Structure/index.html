    Data Structure

    A data structure is a specialized format for organizing, storing, and managing data efficiently. It defines how data is arranged, accessed, and manipulated within a program or system. Data structures are fundamental to computer science and programming, influencing performance, memory usage, and algorithm efficiency.

    Key Characteristics:

    • Organization:
      Defines how data elements relate to each other.
    • Storage:
      Determines how data is physically or logically stored.
    • Access Methods:
      Specifies how data can be retrieved, inserted, updated, or deleted.
    • Efficiency:
      Impacts time and space complexity for operations.

    Linear Data Structures

    • Array (Programming):
      A collection of elements identified by index or key, with fixed size.
    • Linked List:
      A linear collection of elements (nodes) where each element points to the next.
      • Singly Linked List
      • Doubly Linked List
      • Circular Linked List
    • Stack:
      A collection that follows the Last In, First Out (LIFO) principle.
    • Queue:
      A collection that follows the First In, First Out (FIFO) principle.
      • Linear Queue
      • Circular Queue
      • Priority Queue
      • Deque (Double-Ended Queue)

    Non-Linear Data Structures

    • Tree:
      A hierarchical structure with nodes connected by edges, often used to represent hierarchical data.
      • Binary Tree
      • Binary Search Tree (BST)
      • AVL Tree (Self-balancing Binary Search Tree)
      • Heap (e.g., Min-Heap, Max-Heap)
      • Trie (Prefix Tree)
      • Red-Black Tree
      • Segment Tree
      • B-Tree
    • Graph:
      A collection of nodes (vertices) and edges connecting pairs of nodes.
      • Directed Graph
      • Undirected Graph
      • Weighted Graph
      • Adjacency List
      • Adjacency Matrix

    Hashing

    • Hash Table:
      A structure that maps keys to values using a hash function to enable fast lookups.
      • Collision Resolution Techniques: Open Addressing, Chaining (Linked List)
      • Hash Map

    Specialized Data Structures

    • Heap:
      A complete binary tree used for implementing priority queues.
    • Bloom Filter:
      A probabilistic data structure used for set membership testing.
    • Disjoint Set (Union-Find):
      A structure used to manage and find disjoint sets efficiently.
    • Sparse Table:
      Used for efficient range queries (especially in constant time).
    • Suffix Tree:
      Used in string processing algorithms, such as searching for patterns.
