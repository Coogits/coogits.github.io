    Exponential Equation

    An exponential equation is an equation in which the variable appears in the exponent.

    • Key Concept
      • Equation involving Exponentiation
      • Variable as Exponent

    Symbols

    • $a,b\quad$ Bases
    • $x\quad$ Variable as exponent in exponential equations
    • $f(x),g(x)\quad$ The exponent can be expressed as a function of $x$ (e.g. $f(x)=x+1$)

    Formulas

    • General Form
      The general form of an exponential equation is: $$a^{f(x)}=b^{g(x)}$$
    • Standard Form
      This can be solved using logarithms if a and b are not easily related. $$a^x=b$$
    • Equations with the Same Base:
      Given and equation: $$a^{f(x)}=b^{g(x)}$$ If the bases are equal and $a>0$, then by the property of exponential functions: $$f(x)=g(x)$$

    Solution Methods

    Equal Bases

    If possible, express both sides as powers of the same base and equate the exponents.

    1. Example:
      Solve the exponential equation: $$2^{x+1}=8$$
    2. Solution:
      Since $8=2^3$, rewriting the equation with the same base: $$2^{x+1}=2^3$$
    3. Solving
      By equating the exponents $$x+1=3\Rightarrow x=2$$

    Logarithms

    When the equation cannot be rewritten with the same base, apply logarithms to both sides.

    1. Example:
      Solve the equation $$3^x=7$$
    2. Solution
      Apply the Logarithm to both side: $$\log(3^x)=\log7$$ Using logarithm properties: $$x\log3=\log7$$
    3. Solving
      Solving for x: $$x=\cfrac{\log7}{\log3}\simeq1.77$$

    Natural Logarithm

    If the base is Eulerâ€™s number $e$, the natural logarithm ($\ln$) is used.

    1. Example:
      Solve the equation $$e^{2x}=10$$
    2. Solution
      Using the Natural Logarithm: $$\ln(e^{2x})=\ln10$$
    3. Solving
      Solving for x: $$\begin{align}2x=\ln10\\x=\cfrac{\ln10}{2}\end{align}$$

    Properties

    • One-to-One Property
      if $$a^x=a^y\quad\text{where }a>0;a\neq1$$ then $$x=y$$
    • Exponential Growth
      $a^x$ represents exponential growth if: $$a>0$$
    • Exponential Decay
      $a^x$ represents exponential Decay if: $$0<a<1$$

    Quadratic Equation

    Some exponential equations can be rewritten as quadratic equations using substitution.
