    Velocity Addition in Galilean Relativity

    In Galilean relativity, the velocity of an object as measured in one reference frame is related to its velocity in another moving reference frame by simple vector addition. This follows directly from the Galilean transformation, assuming absolute time and space.

    Formulas

    • Velocity Addition: If an object has velocity u in a reference frame S', and S' moves with velocity v relative to another frame S, then the velocity u' of the object in S is given by: $$\vec{u'}=\vec{u}+\vec{v}$$
    • Velocity Addition (1D): If motion occurs along the x-axis only: $$u_x'=u_x+v_x$$

    Symbols

    • $\vec u\quad$ is the velocity of the object in frame S'.
    • $\vec v\quad$ is the velocity of S' relative to S.
    • $\vec{u'}\quad$ is the velocity of the object in frame S.

    Assumptions and Limitations

    Assumptions:

    • Absolute time: Time is the same in all reference frames.
    • Linear velocity addition: Velocities simply add up.

    Limitations:

    • Does not apply to objects moving at speeds close to the speed of light.
    • Replaced by Lorentz transformations in special relativity, where time is relative and space-time is unified.
