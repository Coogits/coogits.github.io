    Bitwise Operator (Programming)

    A bitwise operator is an operator that manipulates individual bits within binary numbers. These operators are fundamental in low-level programming tasks such as bit manipulation, flag management, data compression, cryptography, and performance optimization.

    Core

    • Boolean Type (Programming)
    • Boolean Algebra
    • Binary Digit (Bit)

    Common Bitwise Operators

    And

    • Bitwise AND (&)
    • Rule: 1 & 1 = 1, otherwise 0
    • Example
      1100 (12) 
& 1010 (10) 
------------
  1000 (8)

    OR

    • Bitwise OR (|)
    • Rule: 1 | 0 = 1, 1 | 1 = 1, 0 | 0 = 0
    • Example:
      1100 (12) 
| 1010 (10) 
------------
  1110 (14)

    NOT

    • Bitwise NOT (~)
    • Rule: Flips each bit (0 → 1, 1 → 0)
    • Example (8-bit representation):
    ~ 1100 (12)
------------
  0011 (-13 in Two’s Complement)

    XOR

    • Bitwise XOR (^)
    • Rule: 1 ^ 1 = 0, 0 ^ 0 = 0, 1 ^ 0 = 1
    • Example
      1100 (12) 
^ 1010 (10) 
------------
  0110 (6)

    Left Shift

    • Left Shift (<<)
    • Rule: x << n shifts bits left by n positions (multiplies by 2^n)
    • Example:
    00001100 (12)
   << 2
------------
00110000 (48)

    Right Shift

    • Right Shift (>>)
    • Rule: x >> n shifts bits right by n positions (divides by 2^n)
    • Example:
    00001100 (12)
.   >> 2
------------
00000011 (3)

    Summary

    Common Bitwise Operators:

    Operator Symbol Description
    AND & Performs a bitwise AND operation, setting each bit to 1 only if both corresponding bits are 1.
    OR ` `
    NOT ~ Inverts each bit (0 becomes 1, 1 becomes 0), also known as bitwise negation.
    XOR ^ Performs a bitwise exclusive OR, setting each bit to 1 if the corresponding bits are different.
    Left Shift << Shifts bits to the left, introducing zeros from the right, effectively multiplying by powers of 2.
    Right Shift >> Shifts bits to the right. Behavior depends on whether it is logical (zero-fill) or arithmetic (sign-preserving).
