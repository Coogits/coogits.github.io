    Octal System

    The octal system (also called base 8 numeral system) is a positional numeral system that uses eight digits to represent numbers. The digits in the octal system range from 0 to 7, and each position represents a power of 8.

    Symbols

    • $N_8\quad$ Represent a Octal number $b=8$.
    • $N_{10}\quad$ Represent the equivalent decimal number.
    • $n\quad$ Represent the most significative digit position ($n\geq0$).
    • $-m\quad$ Represent the least significative digit position ($m\leq0$).
      • $m=0\quad$ No fractional digits.
      • $m>0\quad$ Fractional digits
    • $i\quad$ Represent the position of the digit.
    • $a_i\quad$ Represent the digit of the number at position $i$ ($0\leq a_i< 8$).
    • $b^i\quad$ Represent the positional weight associated with digit $a_i$.

    Definition

    • Expanded Form (Numeral System):
      A number in the octal system can be expressed with a subscript $_8$: $$N_8=\displaystyle\sum_{i=-m}^na_i\cdot 8^i$$
    • From Octal to Decima:
      Using the Expanded Form of the binary number: $$N_8=\displaystyle\sum_{i=-m}^na_i\cdot 8^i\quad\to\quad N_{10}$$

    Programming

    In most Programming Languages literal octal numbers are represented as:

    // zero before the octal number
int octal = 0200;
