    Set

    Set is a collection of distinct objects called elements.

    Symbols

    • $A,B\quad$ The sets.
    • $x\quad$ The element of the set in the Set-builder notation.
    • $:\;,\;\mid\quad$ Can be read as "such that"; the symbols are equivalent.
    • $x\in\mathbb{Z}\quad$ The condition in the Set-builder notation.
    • $1,8,5,3\quad$ Elements of the set.

    Notation

    • Roster notation or enumeration notation:
      Defines a set by listing its elements between curly brackets, separated by commas: $$A=\{1,8,5,3\}$$
    • Set-builder notation:
      Specifies a set as a selection from a larger set, determined by a condition on the elements: $$B=\{x\mid x\in\mathbb{Z}\}$$ Alternatively can be used the symbols: $$B=\{x:x\in\mathbb{Z}\}$$

    Special Sets

    • Empty Set:
      Is a set without elements: $$A=\emptyset=\{\}$$
    • Universal Set:
      The universal set includes all elements that are of interest in a given context; is often denoted by the symbol $U$: $$U=\{1,2,3,\ldots\}\qquad U=\mathbb{N}$$
    • Sets of Numbers:
      Main sets of numbers: $$\mathbb{N};\quad\mathbb{Z};\quad\mathbb{Q};\quad\mathbb{R};\quad\mathbb{C}.$$
