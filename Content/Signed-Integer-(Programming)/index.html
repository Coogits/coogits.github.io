    Signed Integer

    A signed integer is a numeric data type that can represent both positive and negative whole numbers.
    It typically uses two’s complement representation in modern computing, allowing efficient arithmetic operations.
    The sign is indicated by the most significant bit (MSB), where 0 represents positive numbers and 1 represents negative numbers.

    • Key Concepts
      • Can hold both positive and negative values.
      • Stored in two’s complement format in most programming languages.
      • Has a fixed range based on the number of bits (e.g., 8-bit signed integer: -128 to 127).
      • Overflow occurs when a value exceeds the defined bit range.
      • Can be mathematically represented as a limited Integer Number Set

    Value Range

    • Signed Integer Value Range:
      One bit is reserved for the sign, reducing the maximum value range by half: $$2^{n-1}-1$$ Explicitly: $$\{-2^{n-1},\ldots,-1,1,0\ldots,2^{n-1}-1\}$$ where:
      • $n\quad$ is the total number of bits in the integer.

    Common Sizes

    Bit Width Minimum Value Maximum Value
    8-bit -128 127
    16-bit -32,768 32,767
    32-bit -2,147,483,648 2,147,483,647
    64-bit -9,223,372,036,854,775,808 9,223,372,036,854,775,807

    Special Cases

    Some programming languages do not support integers explicitly:

    • Python:
      Due to it's dynamical-type nature, python don't handle data types directly.
    • JavaScript:
      Uses floating-point (Number type) for all numbers, including integers. Introduces BigInt for true integer arithmetic.

    Syntax

    int myInt = 42; // In C the size depends on the platform
long myLong = 123456789L; // at least 32-bit, often 64-bit on 64-bit systems
short myShort = 32000;  // typically 16-bit integer
char myByte = 127; // 1-byte(8-bit) integer

    int myInt = 42;  // In C++ the size depends on the platform
long myLong = 123456789L; // at least 32-bit, often 64-bit on 64-bit systems
short myShort = 32000;  // typically 16-bit integer
int8_t myByte = 127; // 8-bit (C/C++ specific, from stdint.h).

    int myInt = 42;  // fixed size 32-bit in C# unlike in C/C++
long myLong = 123456789L; // fixed size 64-bit
short myShort = 32000; // fixed size 16-bit
byte myByte = 255; // 8-bit

    int myInt = 42;
long myLong = 123456789L;
short myShort = 32000;
byte myByte = 127;

    # Python matches the integer based on the value; the size is dynamic.
myInt = 42  
a_bigger_integer = 100000
notInt = 1.1

    // JavaScript has only one number type (double-precision float)
let myInt = 42; 

    var myInt int = 42
var myLong int64 = 123456789
var myShort int16 = 32000

    let my_int: i32 = 42;
let my_long: i64 = 123456789;
let my_short: i16 = 32000;

    var myInt: Int = 42
var myLong: Int64 = 123456789
var myShort: Int16 = 32000

    val myInt: Int = 42
val myLong: Long = 123456789L
val myShort: Short = 32000

    // PHP dynamically determines the type
$myInt = 42; 
```
