    Intersection (Set Theory)

    Intersection: In Set Theory, the intersection of two sets $A$ and $B$, denoted $A\cap B$, is the set of all elements that are in both $A$ and $B$.
    The concept is valid for multiple sets.

    In the image:
    Venn Diagram about a intersection of two sets ($A\cap B$)

    Symbols

    • $\cap\quad$ Intersection Symbol.
    • $A,B,\ldots\quad$ The sets.
    • $\land\quad$ Logic And.

    Formulas

    • Basic Definition:
      The intersection between multiple sets is another set with only the common elements to all sets; is represented with $\cap$: $$A\cap B\cap\ldots=\{x:x\in A \land x\in B\land\ldots\}$$
    • Capital Sigma Notation:
      The intersection of more than two sets (generalized intersection) can be written as: $$\displaystyle\bigcap_{i=1}^{n}A_{i}$$
