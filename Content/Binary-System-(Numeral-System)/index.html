    Binary System

    Binary System is a Numera System uses only two digits (0 and 1).

    Symbols

    • $N\quad$ Is the positional notation of a binary number (e.g. 10110)
    • $a_i\quad$ Binary Digit (Bit), each digit can be 0 or 1 ($0\leq a_i<1$)
    • $n\quad$ Number of non-fractional digits
    • $m\quad$ Number of fractional digits

    Notation

    • Base Notation:
      A binary number can be represented with the number ($N$) followed by the pedice $_2$: $$N_2$$
    • Expanded Form (Numeral System):
      Can be also represented as a polynomial: $$N_2=\displaystyle\sum_{i=-m}^na_i\cdot 2^i\qquad 0\leq a_i<2$$

    Properties

    • Least significative digit:
      If the least significative bit is $0$ the number is a even number
      • $101\underline0\quad$ Even.
      • $010\underline1\quad$ Odd.
