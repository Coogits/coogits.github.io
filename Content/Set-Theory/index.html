    Set Theory

    Set Theory is the branch of mathematical logic that studies sets, which are collections of objects. It provides a foundational framework for modern mathematics by formalizing concepts such as membership, subsets, unions, intersections, and complements. Set theory is not only concerned with finite collections but also with infinite sets, leading to important ideas about cardinality (the notion of the "size" of a set) and ordinals. It underpins virtually all mathematical disciplines by offering a common language and a set of basic principles from which more complex mathematical structures can be built.

    Arguments

    Concepts

    • Set:
      A collection of distinct objects called elements.

    • Element:
      An object that belongs to a set.

    • Sets of Numbers:
      Main sets of numbers.

    • Cardinality:
      $\lvert A\rvert$ The cardinality of a set $A$ is a measure of its size, specifically the number of elements it contains.

    Operators

    • Union:
      ($A\cup B$) Is the set of all elements that are in $A,\;B$, or both.

    • Intersection:
      ($A\cap B$) Is the set of all elements that are in both $A$ and $B$.

    • Complement:
      ($A^\complement$) Is the set of all elements in the universal set $U$ that are not in $A$.

    • Relative Complement:
      ($A\setminus B$) is the set of all elements that are members of $A$ but not members of $B$.

    • Symmetric Difference:
      ($A\triangle B$) Is the set of elements that belong to either $A$ or $B$, but not to both.

    Properties

    • De Morgan's laws:
      First law: $(A\cap B)^\complement=A^\complement\cup B^\complement$. Second law $(A\cup B)^\complement=A^\complement\cap B^\complement$.

    • Venn Diagram:
      Is a visual representation that uses overlapping circles or other shapes to show the logical relationships between two or more sets of items.
