    Linked List

    A linked list is a linear data structure in which elements (called nodes) are stored in separate memory locations and connected through pointers. Unlike arrays, linked lists do not require contiguous memory allocation, making dynamic memory management more efficient.

    Key Concepts

    • Nodes:
      Each node contains:
      • Data: The actual value stored in the node.
      • Pointer: A reference (or link) to the next node in the sequence.
    • Head:
      The first node in the list.
    • Tail:
      The last node, which typically points to null (or nullptr in C++).
    • Dynamic Size:
      Can grow or shrink in size without reallocating memory.

    Types

    Types of Linked List:

    Singly Linked List (SLL)

    A Singly Linked List is a linear data structure where each node points to the next node in the sequence, and the last node points to NULL.

    • Each node has a pointer to the next node.
    • Example:
    [Data | Next] → [Data | Next] → [Data | NULL]

    Doubly Linked List (DLL)

    A Doubly Linked List is similar to a singly linked list but allows traversal in both directions since each node has an additional pointer to the previous node.

    • Each node has pointers to both the next and previous nodes.
    • Example:
    [NULL | Data | Next] ⇄ [Prev | Data | Next] ⇄ [Prev | Data | NULL]

    Singly Circular Linked List (SCLL)

    A Singly Circular Linked List or simply Circular Linked List is a variation of the linked list where the last node is linked back to the first node, forming a circular structure.

    • The last node links back to the first node, forming a loop.
    • Example:
    [Data | Next] → [Data | Next] → [Data | Next] → 
↑                                             ↓
(Head) ←--------------------------------← (Tail)

    Doubly Circular Linked List (DCLL)

    Doubly Circular Linked List is a circular linked list where each node has both prev and next pointers.

    • Example
    ⇄ [Prev|Data|Next] ⇄ [Prev|Data|Next] ⇄ [Prev|Data|Next] ⇄
⇅                                                         ⇅
(Head) ⇄-------------------------------------------⇄ (Tail)

    Comparison

    Type Memory Overhead Traversal Insertion/Deletion Use Cases
    Singly Linked List (SLL) Low Forward only O(1)[1]Time Complexity O(1) at head, O(n)[2][[Time Complexity#O(n) - Linear Time
    Doubly Linked List (DLL) Moderate Forward & Backward O(1) at head/tail, O(n) at middle Undo/Redo, LRU caches
    Circular Linked List (CLL) Low (SLL), Moderate (DLL) Continuous traversal O(1) at head, O(n) at middle Round Robin scheduling, cyclic buffers

    Implementation

    A simple implementation of a singly linked list in C++ with basic operations:

    Features:

    • Insert at the beginning (pushFront)
    • Insert at the end (pushBack)
    • Delete a node (deleteNode)
    • Search for a value (search)
    • Display the list (display)
    #include <iostream>

class LinkedList {
private:
    struct Node {
        int data;
        Node* next;
        Node(int value) : data(value), next(nullptr) {}
    };
    Node* head;
public:
    LinkedList() : head(nullptr) {}
    // Insert at the beginning
    void pushFront(int value) {
        Node* newNode = new Node(value);
        newNode->next = head;
        head = newNode;
    }
    // Insert at the end
    void pushBack(int value) {
        Node* newNode = new Node(value);
        if (!head) {
            head = newNode;
            return;
        }
        Node* temp = head;
        while (temp->next) {
            temp = temp->next;
        }
        temp->next = newNode;
    }
    // Delete a node
    void deleteNode(int value) {
        if (!head) 
	        return;
        if (head->data == value) {
            Node* temp = head;
            head = head->next;
            delete temp;
            return;
        }
        Node* temp = head;
        while (temp->next && temp->next->data != value) {
            temp = temp->next;
        }
        if (temp->next) {
            Node* nodeToDelete = temp->next;
            temp->next = temp->next->next;
            delete nodeToDelete;
        }
    }
    // Search for a value
    bool search(int value) {
        Node* temp = head;
        while (temp) {
            if (temp->data == value)
                return true;
            temp = temp->next;
        }
        return false;
    }
    // Display the list
    void display() {
        Node* temp = head;
        while (temp) {
            std::cout << temp->data << " -> ";
            temp = temp->next;
        }
        std::cout << "NULL\n";
    }
    ~LinkedList() {
        while (head) {
            Node* temp = head;
            head = head->next;
            delete temp;
        }
    }
};
// Example usage
int main() {
    LinkedList list;
    list.pushBack(1);
    list.pushBack(2);
    list.pushBack(3);
    list.display();  // Output: 1 -> 2 -> 3 -> NULL
    list.pushFront(0);
    list.display();  // Output: 0 -> 1 -> 2 -> 3 -> NULL
    list.deleteNode(2);
    list.display();  // Output: 0 -> 1 -> 3 -> NULL
    std::cout << "Search 3: " << (list.search(3) ? "Found" : "Not Found") << "\n";
    std::cout << "Search 5: " << (list.search(5) ? "Found" : "Not Found") << "\n";
    return 0;
}

    Advantages

    • Efficient Insertions/Deletions:
      O(1) complexity at the beginning/middle without shifting elements.
    • Flexible Memory Allocation:
      Does not require pre-allocation like arrays[3]Arrays are faster and require less memory but are less flexible..

    Disadvantages

    • Higher Memory Overhead:
      Each node requires extra space for pointers.
    • Slower Element Access:
      O(n)[2]Linear Time Complexity O(n) time complexity for searching, compared to O(1)[1]Constant Time Complexity O(1) for arrays[3]Arrays are faster and require less memory but are less flexible..
