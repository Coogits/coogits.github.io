    Summation

    A summation is the process of adding a sequence of numbers or quantities together to find their total or sum. It is a fundamental operation in mathematics and is represented symbolically using the plus sign (+) or the summation symbol ($\sum$).

    Symbols

    • $\sum\quad$ Summation Symbol
    • $a_i\quad$ Is the element of the serie identified by the index $i$.
    • $b_j\quad$ Is the element of the serie identified by the index $j$.
    • $i,j\quad$ Is the index of the element in the serie.
    • $m\quad$ The index of the first element. It is often set to $m=1$, and sometimes omitted when the summation starts naturally from 1.
    • $n\quad$ The index of the last element in the series, sometimes omitted if the upper limit is clear from context.
    • $a_{tot}\quad$ is the sum of all the elements ( $a_{tot}=a_m+a_{m+1}+\ldots+a_{n-1}+a_n$ )

    Notation

    • Summation Notation:
      In mathematics, summation is often expressed using the Greek capital letter sigma: $$a_{tot}=\displaystyle\sum_{i=m}^na_i$$
    • Summation Meaning:
      The meaning of summation notation explicitly written out: $$\displaystyle\sum_{i=m}^na_i=a_m+a_{m+1}+\ldots+a_{n-1}+a_n$$
    • Summation with multiple index:
      This notation signifies summation over all distinct pairs $i,j$ where $i$ is less than $j$. This ensures that each unique pair is considered only once, preventing redundant calculations.: $$\sum_{i<j}(a_i,b_j)$$

    Programming

    In Programming Language, a summation can be represented as the accumulation of array elements (or values) over a specified range.

    const int m = 0; // Start index of the summation range
int a[] = {0, 1, 2, 3}; // Array of values to sum
const int n = sizeof(a)/sizeof(a[0]); // Number of elements in the array
int summation = 0; // Initialize the summation variable

// Iterate through the array from index m to n-1
for(int i = m; i<n; i++){
	summation += a[i]; // Add each element of the array to the summation variable
}

    In the code above the summation can be expressed as:

    • $\displaystyle\sum_{i=m}^na_i\qquad \text{where}\begin{cases}m=0\\ n=4\\ a_i=i\end{cases}$
    • $\displaystyle\sum_{i=0}^4i=0+1+2+3=6$
