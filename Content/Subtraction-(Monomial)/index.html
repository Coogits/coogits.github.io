    Subtraction (Monomial)

    Monomials can only be subtracted if they are like terms, that is, they have the same variables raised to the same exponents. If the variables or exponents differ, they cannot be combined by subtraction.

    Formulas

    • Rule:
      When subtracting like monomials, subtract their coefficients while keeping the variable part unchanged: $$ax^{k_x}y^{k_y}\ldots-bx^{k_x}y^{k_y}\ldots=(a-b)x^{k_x}y^{k_y}\ldots$$

    Examples

    • Like Terms: $$3xy-2xy\quad = \quad xy$$
    • Like Terms: $$4x^2y-2x^2y\quad = \quad 2x^2y$$
    • Not Like Terms (Different Exponents):
      Cannot be subtracted because the exponents of x differ $${\color{gray}4x^3-5x^2}$$
    • Not Like Terms (Different Variables):
      Cannot be subtracted because the variables differ $${\color{gray}4x^2-5y^2}$$

    Limitations

    • Like Terms Requirement:
      Monomials must share the exact same variables with the exact same exponents to be subtracted. Any difference in these elements means the monomials are not like terms and cannot be combined through subtraction.
